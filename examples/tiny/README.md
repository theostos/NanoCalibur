# Tiny Runnable Examples

Each script is self-contained and can be run directly:

```bash
python examples/tiny/01_minimal_game.py
python examples/tiny/02_input_phases.py
python examples/tiny/03_safe_collision.py
python examples/tiny/04_tool_call.py
python examples/tiny/05_local_role_fields.py
python examples/tiny/06_cameras.py
python examples/tiny/07_turn_based.py
python examples/tiny/08_role_interface.py
```

All scripts compile in-memory DSL source and print a short summary.
